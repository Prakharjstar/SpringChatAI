About This Chat App

A modern AI-powered chat application built using Spring Boot, Thymeleaf, TailwindCSS, and Llama2 AI integration via Ollama.

Tech Stack

Spring Boot

Backend framework for building robust Java applications.

Thymeleaf

Template engine to render dynamic HTML pages.

TailwindCSS

Modern utility-first CSS framework for beautiful UIs.

Ollama / Llama2 AI

AI integration to power smart responses in real-time chat.

HTML & CSS

Front-end structure and styling for a responsive layout.

AI Integration

This chat app uses Llama2 AI via Ollama API to generate smart responses instantly. Combined with Spring Boot for the backend and Thymeleaf templates for rendering, it delivers a modern, responsive, and engaging user experience.
